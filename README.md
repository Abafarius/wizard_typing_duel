# Wizard Typing Duel — Step 2

Маленькая pygame-игра в духе старых flash-аркад: печатай слова на летящих заклинаниях, пока они не попали в мага.

## Запуск

```powershell
venv\Scripts\activate
python wizard_typing_duel.py
```

## Управление

- В меню: `1` Easy, `2` Normal, `3` Hard
- `Space` / `Enter` — старт или рестарт
- Печатай слово на заклинании, чтобы уничтожить его
- `Backspace` — удалить символ
- `Enter` — очистить ввод
- `Esc` — выйти

## Что добавлено в Step 2

- Баланс вынесен в `DIFFICULTIES`
- 3 сложности: Easy / Normal / Hard
- Новые списки слов: короткие, средние, сложные фразы, боссовые фразы
- Уровни влияют на скорость и частоту появления заклинаний
- Ограничение максимального числа заклинаний на экране
- Комбо и бонус за комбо
- Босс каждые N уничтоженных заклинаний
- Разные цвета обычных, сложных и боссовых заклинаний
- Подсветка заклинания, которое совпадает с текущим вводом

## Где крутить баланс

Открой `wizard_typing_duel.py` и смотри блок `DIFFICULTIES` сверху.

Самые важные параметры:

- `health` — сколько HP у игрока
- `base_spawn_delay` — начальная задержка между заклинаниями
- `min_spawn_delay` — минимальная задержка на высоких уровнях
- `speed_min` / `speed_max` — базовая скорость заклинаний
- `speed_per_level` — насколько ускоряется игра с каждым уровнем
- `max_spells` — максимум заклинаний на экране
- `level_every` — через сколько уничтоженных заклинаний растет уровень
- `boss_every` — через сколько уничтоженных заклинаний появляется босс
