# Wizard Typing Duel — MVP

Мини-флеш-игра на Python + pygame.

## Запуск

```bash
python -m venv venv
venv\Scripts\activate      # Windows PowerShell
# source venv/bin/activate  # macOS/Linux
pip install pygame
python wizard_typing_duel.py
```

## Управление

- Печатай слово над летящим заклинанием, чтобы уничтожить его.
- Backspace — удалить символ.
- Enter — очистить ввод.
- Esc — выйти.
